Intro to Markdown

Each slide shows Markdown, HTML, and a rendered view.

Use ⬅️ ➡️ arrows or the buttons below to navigate.

Headings

One of each heading level (H1–H3 shown).
Markdown
# Title H1
## Section H2
### Subsection H3
HTML
<h1>Title H1</h1>
<h2>Section H2</h2>
<h3>Subsection H3</h3>
Rendered

Emphasis

Italic, bold, and strikethrough.
Markdown
*italic*  **bold**  ~~strikethrough~~
HTML
<em>italic</em>  <strong>bold</strong>  <del>strikethrough</del>
Rendered

Paragraphs & Line Breaks

Blank line for new paragraph; two spaces at end for a line break.
Markdown
First paragraph line one␠␠
line two (same paragraph)

Second paragraph
HTML
<p>First paragraph line one<br>
line two (same paragraph)</p>
<p>Second paragraph</p>
Rendered

Unordered List

One top-level item and a nested item.
Markdown
- Item A
  - Nested A.1
HTML
<ul>
  <li>Item A
    <ul>
      <li>Nested A.1</li>
    </ul>
  </li>
</ul>
Rendered

Ordered List

A simple numbered list.
Markdown
1. Step one
2. Step two
HTML
<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>
Rendered

Links

Basic hyperlink syntax.
Markdown
[OpenAI](https://openai.com)
HTML
<a href="https://openai.com">OpenAI</a>
Rendered

Images

Alt text is required for accessibility.
Markdown
![Alt text](https://example.com/logo.png)
HTML
<img src="https://example.com/logo.png" alt="Alt text">
Rendered

Code

Inline and fenced code blocks.
Markdown
Inline: `const x = 1`

```js
function hello() {
  console.log('hi');
}
```
HTML
Inline: <code>const x = 1</code>

<pre><code class="language-js">function hello() {
  console.log('hi');
}
</code></pre>
Rendered

Blockquotes

Quotes can nest.
Markdown
> A wise thought
>> A nested idea
HTML
<blockquote>
  <p>A wise thought</p>
  <blockquote>
    <p>A nested idea</p>
  </blockquote>
</blockquote>
Rendered

Tables

Alignment via colons in the divider row.
Markdown
| Col A | Col B |
|-----:|:-----:|
|  123 |  xyz  |
HTML
<table>
  <thead>
    <tr><th style="text-align:right">Col A</th><th style="text-align:center">Col B</th></tr>
  </thead>
  <tbody>
    <tr><td style="text-align:right">123</td><td style="text-align:center">xyz</td></tr>
  </tbody>
</table>
Rendered

Task List (GFM)

Rendered as disabled checkboxes on GitHub.
Markdown
- [ ] Todo item
- [x] Done item
HTML
<ul>
  <li><input type="checkbox" disabled> Todo item</li>
  <li><input type="checkbox" disabled checked> Done item</li>
</ul>
Rendered
